# Proyecto Katas de Python

## Descripción

Proyecto de lógica y programación en Python realizado como parte del máster en Data Analytics de ThePower. El objetivo del proyecto ha sido resolver una colección de 41 katas de Python para practicar conceptos fundamentales del lenguaje, incluyendo funciones, programación funcional, manejo de excepciones, clases, recursividad y estructuras de control.

A lo largo del proyecto se han implementado soluciones sencillas y comentadas para facilitar la comprensión de los conceptos trabajados en clase.

## Estructura del proyecto

```text
katas_python/

├── katas.py       # Resolución de las 41 katas de Python
├── README.md      # Documentación del proyecto
```

## Conceptos trabajados

- Funciones y funciones lambda.
- Estructuras de control (`if`, `elif`, `else`).
- Bucles (`for`).
- Listas, tuplas y diccionarios.
- Programación funcional con `map()`, `filter()` y `reduce()`.
- Manejo de excepciones (`try`, `except`).
- Recursividad.
- Programación orientada a objetos.
- Clases y métodos.
- Manipulación de cadenas de texto.

## Resultados y conclusiones

Durante el desarrollo del proyecto se han resuelto un total de 41 ejercicios prácticos que abarcan distintos niveles de dificultad.

Entre los principales aprendizajes destacan:

- Uso de funciones para modularizar el código.
- Aplicación de programación funcional mediante `map()`, `filter()` y `reduce()`.
- Creación y gestión de excepciones personalizadas.
- Implementación de algoritmos recursivos.
- Desarrollo de clases y objetos en Python.
- Manipulación y procesamiento de listas, cadenas y diccionarios.

## Retos encontrados

- Comprender el funcionamiento de `reduce()` y la librería `functools`.
- Trabajar con funciones lambda.
- Implementar la recursividad en el cálculo del factorial.
- Diseñar clases y métodos para modelar objetos como árboles y usuarios bancarios.
- Gestionar excepciones y errores introducidos por el usuario.
